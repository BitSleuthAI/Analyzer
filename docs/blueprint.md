# **App Name**: BitSleuth

## Core Features:

- Connect Wallet: Securely connect a Bitcoin wallet by entering the xpub key.
- Dashboard Overview: Display a dashboard summarizing wallet balance, transaction history, and a security/health score.
- Transaction Details: Provide a detailed view of individual transactions, including ID, date, amount, addresses, and confirmation status.
- Security Analysis: Assess and display the wallet's security status, including opsec threat level, reused addresses, and dust.
- AI Chat Interface: Implement an AI-powered chat interface for users to query the tool about their wallet using natural language, with AI responses providing insights and explanations.

## Style Guidelines:

- Primary color: Electric Purple (#BF5FFF), offering a high-tech, secure, and slightly mysterious feel, while hinting at the decentralized nature of Bitcoin.
- Background color: Off-Black (#1A1A1A), providing a sleek, modern dark mode that reduces eye strain and enhances the vibrancy of UI elements.
- Accent color: Vivid Pink (#FF69B4), used to highlight key interactive elements and call-to-action buttons for intuitive navigation.
- Body and headline font: 'Inter', a grotesque-style sans-serif, provides a modern and neutral look suitable for clear and concise information presentation.
- Use clean, minimalist vector icons that clearly represent different aspects of Bitcoin transactions and wallet security.
- Implement a modular layout with clear visual hierarchy, making it easy for users to understand their wallet information and navigate the app.
- Incorporate subtle, non-intrusive animations for transitions and loading states to improve user experience without distracting from the app's core functionality.
